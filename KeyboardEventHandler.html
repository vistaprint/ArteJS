Keyboard Event Handler

Extends Arte.Text area to listen to and handle keyboard events

Supported Events

The keyboard event handler adds support for the following:

*Key Event* *Action*
CTRL+B Bold
CTRL+I Italic
CTRL+U Underline
CTRL+A, CTRL+V, All Arrow Keys Fires onselectionchange event
